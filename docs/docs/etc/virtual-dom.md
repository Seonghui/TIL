Virtual Dom은 DOM 트리를 추상화해서 JavaScript 객체로 만들어 둡니다. 변경점은 가상돔에서 처리하고 실제 돔 조작은 최소화해서 브라우저의 리플로우나 리페인트 횟수를 현저히 줄이는 방식으로 퍼포먼스를 높인 것입니다.
