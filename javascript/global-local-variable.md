# 전역 변수와 지역 변수

## 1. 전역 변수 \(Global Variable\)

* 자바스크립트에서 제일 바깥 단위에 선언하는 변수.
* 함수에 포함되지 않는다.
* 어디에서나 사용이 가능하다.
* 사용을 자제하는 게 좋다. \(보안 등의 이유\)

## 2. 지역 변수 \(Local Variable\)

* 함수 내부에 선언된 변수로, 함수가 실행되면 만들어지고 함수가 종료되면 소멸하는 변수이다.
* 함수 외부에서는 접근할 수 없다.

## 3. 예제

```javascript
var a = "전역 변수";

function myFunc() {
  var b = "지역 변수";
  console.log(b);
}

myFunc();
console.log(a);
```

여기서 a 는 전역 변수, b 는 지역 변수이다.
